INSTRUCTIONS POUR LES IMAGES D'EN-TÊTE DU PORTFOLIO

J'ai modifié les chemins d'accès des images d'en-tête pour toutes les pages de votre portfolio afin qu'elles utilisent des images représentant des personnes à peau noire, correspondant à votre profil.

Pour compléter cette personnalisation, veuillez ajouter les images suivantes dans ce dossier :

1. contact-header.jpg - Une image représentant une personne noire dans un contexte professionnel de communication (par exemple, au téléphone, devant un ordinateur, etc.)

2. portfolio-header.jpg - Une image montrant une personne noire travaillant sur un projet créatif ou présentant son travail

3. services-header.jpg - Une image d'une personne noire offrant un service ou dans un contexte de consultation professionnelle

4. skills-header.jpg - Une image d'une personne noire démontrant des compétences techniques (programmation, design, etc.)

Pour la page d'accueil et la page À propos, j'ai déjà configuré l'utilisation de votre photo personnelle (profile.jpg).

Dimensions recommandées pour les images d'en-tête :
- Largeur : au moins 1920px
- Hauteur : au moins 1080px
- Format : JPG ou PNG
- Taille de fichier : optimisée pour le web (moins de 500 Ko si possible)

Ces images seront affichées avec un filtre d'assombrissement pour assurer la lisibilité du texte qui apparaît par-dessus.
