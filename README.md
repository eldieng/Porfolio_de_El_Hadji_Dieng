# Portfolio Professionnel d'El Hadji Dieng

Ce projet est un portfolio professionnel pour El Hadji Dieng, Web Master sénégalais, développé avec React.

## Présentation

Ce portfolio a été conçu pour mettre en valeur les compétences, réalisations et services d'El Hadji Dieng. Il vise à renforcer son image professionnelle, faciliter la prise de contact avec des recruteurs ou des clients potentiels et asseoir sa présence digitale.

## Fonctionnalités

- Design moderne et responsive
- Animations et transitions fluides avec Framer Motion
- Navigation intuitive
- Présentation des projets et compétences
- Section blog avec articles et filtres par catégorie/tag
- Formulaire de contact
- Option multilingue (français/anglais)
- Optimisation SEO et données structurées
- Code splitting et lazy loading pour des performances optimales

## Structure du site

- **Accueil** : Présentation générale avec animation de texte
- **À propos** : Biographie et parcours professionnel
- **Compétences** : Présentation des compétences techniques
- **Portfolio** : Galerie des projets réalisés (sites web, UI/UX, applications web, refonte)
- **Services** : Description des services proposés
- **Blog** : Articles, tutoriels et ressources sur le développement web et design
- **Témoignages** : Avis de clients satisfaits
- **Contact** : Formulaire de contact et coordonnées

## Technologies utilisées

- React 18+
- React Router v6 pour la navigation
- Styled Components pour le styling
- Framer Motion pour les animations
- React Icons pour les icônes
- React Helmet Async pour la gestion du SEO
- React.lazy et Suspense pour le code splitting
- React Error Boundary pour la gestion des erreurs
- Google Analytics pour le suivi des performances

## Installation et démarrage

1. Cloner le dépôt :
```
git clone [URL_DU_REPO]
```

2. Installer les dépendances :
```
cd portfolio-el-hadji-dieng
npm install
```

3. Lancer l'application en mode développement :
```
npm start
```

4. Construire l'application pour la production :
```
npm run build
```

## Personnalisation

Pour personnaliser ce portfolio :

1. Modifiez les informations personnelles dans les composants
2. Ajoutez vos propres projets dans le composant PortfolioSection
3. Ajoutez vos propres articles de blog dans `blogData.js`
4. Remplacez les images dans le dossier `/public/assets/images`
5. Personnalisez les couleurs et polices dans `GlobalStyles.js` et `Theme.js`
